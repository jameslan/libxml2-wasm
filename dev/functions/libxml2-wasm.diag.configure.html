Function configure

  • configure(options: DiagOptions): void

  • Set up memory diagnostic helpers.

    When enabled, these helpers will record allocated XmlDisposable objects (and their subclass objects) and track whether the dispose method is called.

    Note that the allocation of these objects will not be monitored before memory diagnostics is enabled.
